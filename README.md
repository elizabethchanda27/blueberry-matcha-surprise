# Blueberry Matcha Surprise

React (Vite) birthday intro: blueberry-matcha swirl, “Happy Birthday” + cake animation, then a link to your app.

## Run

```bash
npm install
npm run dev
```

Open the URL Vite prints (e.g. http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

Output is in `dist/`. Deploy that folder to any static host.

## Point to your app

Edit `src/App.jsx`: change the `APP_URL` constant to your app’s URL (e.g. `'https://yoursite.com'` or `'/app'`).

## Font

- **Current:** Fredoka. To use d-wardrobe style (Gaegu), add the Gaegu font link in `index.html` and set `font-family: 'Gaegu', cursive` in `src/index.css` for `body`.
